# ArtTech Backoffice Platform

미술품 투자 서비스를 위한 Django 기반 풀스택 백오피스 플랫폼입니다.  
관리자 및 투자자 관점에서 미술품 관리, 투자 현황 조회, 통계 대시보드를 제공합니다.

---

## 🔧 Tech Stack

### Backend
- Python 3.11
- Django, Django REST Framework
- PostgreSQL

### Frontend
- Django Template
- HTML / CSS
- JavaScript (jQuery)

### DevOps
- Docker, Docker Compose
- AWS ECS (Fargate)
- CircleCI (CI/CD)

### Collaboration
- Bitbucket (Git Flow)
- Jira (Agile Sprint)
- Confluence (문서화)

## ✨ 주요 기능

### 1. 사용자 / 권한 관리
- Custom User 모델 사용
- 관리자 / 일반 투자자 권한 분리
- Django 인증 시스템 활용

### 2. 미술품 관리
- 미술품 CRUD
- 가격 및 작가 정보 관리
- 관리자 전용 관리 화면

### 3. 투자 기능
- 사용자별 미술품 투자
- 투자 금액 기록
- 투자 내역 조회

### 4. 관리자 대시보드
- 전체 미술품 수
- 전체 투자 건수
- 관리자 전용 접근 제어
